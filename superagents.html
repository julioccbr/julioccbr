Superagents - Web AI Chat App

Role

Product Designer / Low Code Dev

Timeline

3 Months

Platform

Web App (Desktop)

Tools

Figma, Miro, N8N, WhatsApp AI Hub, Meta, Webflow

The Context

Superagents is an AI-powered conversational platform built to automate customer scheduling and support for service-based businesses (clinics, salons, procedure bookings). By bridging the gap between automated chat and human agency, we designed a tool that allows customers to have natural conversations while automatically managing bookings. The platform reached a key industry milestone by becoming an official Meta Tech Provider and integrating directly into the WhatsApp AI Hub.

The Goal: The project aim was to build the frontend platform and automate workflows for Superagents. Our primary goal was to bypass the rigid, numeric option menus of legacy bots, introducing fluid, human-like AI conversations that auto-book appointments directly onto clients' Google Calendars and enable a seamless transition to human agents.

The Challenge

01.

Rigid Legacy Chatbots

Traditional bots restricted users to rigid menu flows (e.g., "press 1 for X, 2 for Y"), blocking natural human conversation, failing to resolve custom queries, and generating immense user friction.

02.

Disconnected Scheduling

Legacy automated chat solutions could converse but were completely disconnected from business tools. Agents were forced to manually verify schedules, copy details, and book slots, causing delays and double bookings.

03.

High Handover Friction

Transitioning a conversation from an automated AI bot to a live human support agent lacked clear UI indicators, leaving customers confused or unaware of the switch.

The Solution

01.

Open Conversational Flows

Designed and built flexible, open-ended conversational interfaces. This allowed users to converse naturally, ask specific questions, and receive human-like contextual responses instead of being locked into standard number pads.

02.

N8N Scheduling Engine

Built robust N8N backend workflows that automated appointment scheduling. The bot directly booked available slots and automatically synchronized them with the client's Google Calendar, completely eliminating manual copy-pasting.

03.

Dual Inbox & Handover UI

Designed a desktop dashboard separating bot-managed chats from the human agent queue. A graceful alert notified the user that a human agent would soon assist them, ensuring transparent handovers.

04.

Bespoke Visual UI & Webflow Development

Designed the entire visual identity from scratch without relying on a pre-existing component library or design system. This visual freedom allowed for a highly tailored experience that I manually built and deployed as a responsive frontend using Webflow.

The Results

The Superagents platform successfully automated scheduling and customer interaction. By combining N8N workflows with a clean desktop inbox, the project secured a Meta Tech Provider status and integration into the WhatsApp AI Hub.
